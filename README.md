# Atom Snippets

Code snippets for Atom.
In order to let you develop faster. 

为Atom开发提供snippets功能
Atom Snippets的使用需要配合Mota插件提供的语言定义
所以使用前请确保安装并启动了Mota插件

## Features

- 提供Atom模板部分常用模板的Snippets功能
- 提供快捷键（⌘+.）跳转到索引平台

## Plans

- 对全部Atom组件的Snippets功能支持
- Ala基础函数库Snippets
- 鼠标悬浮+快捷键show组件基本用法
- 自动添加config中对组件的引用
